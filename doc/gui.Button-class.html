Method Details

__init__(self, image, action, x, y, text='', parent_group=None, more_draw=None)
(Constructor)

 
Parameters:
  • image - Button background image
  • action - Function to call when pressed
  • x, y - Bottom left corner position
  • text - Label text
  • parent_group - ButtonGroup that owns this object.
  • more_draw - Function called immediately after the button is drawn. Use if you want to draw more stuff on top of the button other than an image. See tool.generate_brush_selector() for an example.

select(self)

 

After making a button group, call one button's select() method to make it the default. The button's action is not called.
