Module tool

Base class and utilities for tools.

Writing a Tool

  1. Subclass Tool
  2. Override Appropriate Methods

    Most simple tools will want to at least override start_drawing() and keep_drawing(). More sophisticated tools will need to override more methods to achieve the desired behavior.

  3. Set Module Properties : 
       default = YourClass()   #: Instance of your class
   priority = 1000         #: Rank
   group = 'Example'       #: Grouping - Drawing, Shapes, etc
   image = None            #: Icon
   cursor = None           #: Default cursor

Adding Buttons

  1. Make a button. See Button and ImageButton.
  2. Add it to the control space with tool.controlspace.add().

Function Details

generate_brush_selector(start_x=5, start_y=5, max_x=-1, max_y=-1)

 

Generate a line of buttons that let the user change the brush size. See Brush tool for an example.

Parameters:
  • start_x, start_y - Bottom left corner of the button row
  • max_x, max_y - Maximum values for button positions. Should generally be ignored.
Returns: ButtonGroup
The ButtonGroup that owns the generated buttons
